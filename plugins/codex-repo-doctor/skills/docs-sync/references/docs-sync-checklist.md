# Docs Sync Checklist

- README commands still exist in `package.json`.
- Installation instructions match the package manager and published package names.
- CLI options match the actual parser behavior.
- Generated file lists match current templates.
- Generated skill folders include `references/` files, not only `SKILL.md`.
- Package scripts mentioned in docs still match root and workspace `package.json` files.
- Public exports and examples match source behavior.
- Deprecated APIs are not promoted as normal usage.
- Release notes mention user-visible behavior changes.
